Install Ubuntu 12.04 LTS:

  1. Make a liveUSB with software like Rufus, need to download Ubuntu image first.

  2. Change the boot order of the laptop then boot from the USB stick. For Windows 8, disable the secure boot and may also need to change the boot type for compatibility issues.

  3. Partition the disk:

    • if the space is not enough, resize the disk in windows when necessary.
    • select the primary partition: format - ext4 journal system, mount point - /
    • make one swap partition with size around 5 GB.

  4. Follow all next steps.

Install GNURadio and UHD:

  1. Modify the source by changing the source list:

    sudo gedit /etc/apt/source.list

    Then, copy the following to the top of the list (this will choose the best server for downloading software):

    deb mirror://mirrors.ubuntu.com/mirrors.txt precise main restricted universe multiverse
deb mirror://mirrors.ubuntu.com/mirrors.txt precise-updates main restricted universe multiverse
deb mirror://mirrors.ubuntu.com/mirrors.txt precise-backports main restricted universe multiverse
deb mirror://mirrors.ubuntu.com/mirrors.txt precise-security main restricted universe multiverse

    At last, update the list:

    sudo apt-get update

  2. Download and run script to install GNURadio;

    sudo apt-get -y install git-core autoconf automake  libtool g++ python-dev swig \
pkg-config libboost1.48-all-dev libfftw3-dev libcppunit-dev libgsl0-dev \
libusb-dev sdcc libsdl1.2-dev python-wxgtk2.8 python-numpy \
python-cheetah python-lxml doxygen python-qt4 python-qwt5-qt4 libxi-dev \
libqt4-opengl-dev libqwt5-qt4-dev libfontconfig1-dev libxrender-dev

  3. Build GNURadio and UHD

    use the -o option to download the stable version of GNURadio

    wget http://www.sbrac.org/files/build-gnuradio && chmod a+x ./build-gnuradio && ./build-gnuradio -o

  4. For other version of Ubuntu systems, check this GNURadio page for more information.
